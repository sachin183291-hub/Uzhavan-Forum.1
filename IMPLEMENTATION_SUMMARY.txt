⚡ QUICK START COMMANDS

1. Install dependencies:
   npm install

2. Start development (option A - both servers):
   npm run dev:all

   Or (option B - separately):
   npm run dev              # Frontend on port 3000
   npm run dev:server       # Backend on port 5000

3. Access the application:
   Login:        http://localhost:3000/axis-login
   Dashboard:    http://localhost:3000/smart-irrigation
   API Health:   http://localhost:5000/api/health

📡 API ENDPOINTS (Base: http://localhost:5000/api)

AUTHENTICATION
  POST /auth/login           - Login
  POST /auth/otp             - Verify OTP
  POST /auth/biometric       - Verify biometric
  POST /auth/logout          - Logout

SENSORS
  GET  /hardware/sensors     - Real-time sensor data
  GET  /hardware/status      - System status

MOTOR CONTROL
  POST /hardware/motor/on    - Turn ON
  POST /hardware/motor/off   - Turn OFF
  POST /hardware/motor/auto  - AUTO mode
  POST /hardware/motor/manual- MANUAL mode

CONFIGURATION
  GET  /hardware/config      - Get config
  POST /hardware/config/update - Update config

ALERTS
  POST /alerts/sms           - Send SMS

SYSTEM
  GET  /health               - Health check
